An extremely basic WIN32 application that calculates the correct length for a Kayak paddle on given data. Programmed in DEV C++ with no visual environment.

The tech utilised includes:

  • C++
  • WIN32
  • DEV C++

It was nice to play with C++ again after many years and I found the Embarcadero IDE pretty good to use. Additionally, making an application using code without the advantage of a visual 'Click & Play environment was a good reminder of life before Visual Studio etc.

I would like very much to go back and play again with C++, perhaps do some simple console based programs, sort algorithms maybe? I called the repo 'Random Win App' because I literally had no clue what I was going to create when I started, I just wanted something in C++ and Win 32.

The repository is hosted on GitHub at https://github.com/mxfoyster/randomWinApp
