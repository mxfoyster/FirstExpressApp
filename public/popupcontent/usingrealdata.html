An opportunity to play with some real data. I downloaded some .csv's from Google Trends relating to watersports. Then I used JavaScript and later on some PHP to load and process the data. Chart.js was used to display the information visually. It was my first use of Chart.js, I found it fun.

The tech utilised includes:

  • HTML & CSS
  • JavaScript
  • Chart.JS & PHP

You will see from the website how my understanding and usage of Chart.JS improved as the project progressed.
