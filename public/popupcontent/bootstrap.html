This is a simple website where I played with bootstrap to see what it was like to use. Additionally, I played with PHP and MySQL using a test database I found online. It gave me the opportunity to make a sipmle API to communicate with the database and to construct simple widgets to sort and filter the data

The tech utilised includes:

  • HTML, CSS and Bootstrap
  • JavaScript
  • PHP & MySQL

I found bootstrap quite simple to use, with enough time invested to familiarise myself with it further, it would be simple and fast to construct quite complex websites using it. The biggest downfall is the extra work required for overriding some of the set styles. For websites that are highly visually customised, it may be less efficient initially.

The source code for this website is not on GitHub for security and other personal reasons. Viewing it can be arranged under the appropriate circustances.
