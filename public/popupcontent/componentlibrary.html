I collected some code snippets from various past projects and turned them into a simple component library. The GitHub repository contains basic usage instructions within markdown files. Additionally, I have hosted the wesbite built into the project, giving further instructions and demonstrating each component

The components include the following:

  • Plain HTML & CSS components
  • HTML, CSS, Javascript and some PHP components
  • A gallery viewer with zoom and drag capability

The components are very simple, it's not a complex setup but it can grow in time and alongside being a fun project, I find it handy to look back on for reference purposes whilst doing other projects.

You can visit the website at https://componentlibrary.markfoyster.co.uk/.

The repository is hosted on GitHub at https://github.com/mxfoyster/componentlibrary1
