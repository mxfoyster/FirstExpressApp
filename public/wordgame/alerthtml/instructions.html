Word Game Instructions and Settings

Welcome to my Word Game. This is Hangman with a difference. Guess the name of the movie before a penguin is built to win.

Simply try different letters and see if you get a match. In easy mode, you have 9 attempts. In Hard mode you only have 7. Set the difficulty below:

Wrong guesses are displayed below the buttons, the game will let you choose the wrong letter more than once so be careful.

Stuck? Try the in game button. It gives you a catchphrase or details about the movie but BEWARE, it will cost you TWO GOES. Less that two goes left? sorry, hint will not be available!

Think you know the movie name? Use the button and see if you're correct. But if you're wrong, you'll automatically LOSE so choose carefully!

If you want to read these instructions again, just press the button in the game.
